{\footnotesize\ttfamily byte? Cymatic\+Labs.\+Unity3\+D.\+Amqp.\+Amqp\+Client.\+Subscribe\+Retry\+Limit\hspace{0.3cm}{\ttfamily [get]}, {\ttfamily [set]}}



Gets or sets the maximum number of failed subscriptions the client will tolerate before preventing connection to the host. 

Presently in Rabbit\+MQ if there is an error during subscription the host will close the connection so the client must reconnect.
